## Our Projects

### Taskify: An IDE to Develop and Debug Intermittent Software

```markdown
Taskify is a tool that allows programmers to write and debug task-based intermittent programs in a domain specific language designed for the development of intermittent systems on any general purpose computer. Taskify automatically transforms these programs into C programs that can be linked to the underlying run-time environment and deployed into the target platform. Taskify implemented as an Eclipse plugin.
```
Project website: [Taskify](https://github.com/tinysystems/Taskify)
